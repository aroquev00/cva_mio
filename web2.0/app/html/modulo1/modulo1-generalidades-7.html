Módulo 1: Actividad

Instrucciones:
Lee con atención cada enunciado y selecciona la opción correcta (Verdadero o Falso).




1. Las páginas de la Web 2.0 son estáticas

2. Los blogs y las Wikis forman parte de la Web 2.0.

3. La web 2.0 está basada en comunidades de usuarios que fomentan la colaboración e intercambio de información.

4. El navegar y consumir son características de la Web 1.0.

5. El conectar y colaborar son características de la Web 2.0.

6. La Web 2.0 no es social ni flexible.

7. La Web 2.0 usa tecnología como RSS, Java y AJAX

8. La Web 1.0 no usaba el lenguaje de programación html

9. La Web 2.0 a través de los RSS permite saber las novedades en cada sitio sin necesidad de que el usuario visite el sitio o páginas para saberlo.

10. Las herramientas que se usan en la Web 2.0 no permiten compartir experiencias ni utilizar la experiencia o ideas de otros.
